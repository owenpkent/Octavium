# Octavium v1.1.1 — Build & Distribution Pipeline

**Release Date:** February 7, 2026
**Signed By:** OK Studio Inc. (Sectigo EV Code Signing Certificate)

---

## What's New

### EV Code Signing
- Octavium.exe is now signed with an Extended Validation certificate from OK Studio Inc.
- **No more SmartScreen warnings** — Windows trusts the executable immediately
- Timestamped signature remains valid beyond certificate expiry

### Windows Installer (InnoSetup)
- Professional installer with admin-level installation to Program Files
- **Optional MIDI Chord Library** — choose "Full" install to include 15,000+ chord voicings, or "Compact" for just the application
- Previous version detection with automatic upgrade/downgrade handling
- Desktop and Start Menu shortcuts
- Clean uninstaller

### Build Pipeline
- Automated build → sign → package → sign pipeline (`build_installer.ps1`)
- MIDI chord library auto-fetched from GitHub during build if not present
- Dynamic MIDI library version discovery — compatible with future releases

### Repository Cleanup
- Removed ~15,000 MIDI files from git tracking (30 MB saved per clone)
- MIDI library now fetched on-demand via `scripts/fetch_midi_library.ps1`
- Dynamic directory discovery in `midi_chord_loader.py` — works with any library version

## Requirements

- Windows 10 or later (64-bit)
- Virtual MIDI port (e.g., [loopMIDI](https://www.tobias-erichsen.de/software/loopmidi.html))
- Any DAW that accepts MIDI input

## Building from Source

See [BUILD.md](docs/BUILD.md) for full instructions.

```powershell
git clone https://github.com/owenpkent/Octavium.git
cd Octavium
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
.\scripts\fetch_midi_library.ps1   # optional: download MIDI chord library
python run.py
```
